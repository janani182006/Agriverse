Getting Started

5 questions

What is AGRIVERSE?

AGRIVERSE is a smart farming assistant designed specifically for Sikkim farmers...

How do I set up sensors for my field?

You can install IoT-based soil moisture and temperature sensors that connect automatically to the system.

How do I manage multiple fields?

You can add multiple fields during setup and assign different crops to each.

Is AGRIVERSE suitable for small-scale farmers?

Yes, it is designed especially for small and marginal farmers to make smart farming more affordable and practical.

Do I need internet to use AGRIVERSE?

Only the dashboard and updates require internet. Sensor data is stored and synced automatically when connected.

Irrigation & Monitoring

4 questions

How does the pump control work?

AGRIVERSE triggers irrigation when soil moisture drops below optimal levels.

Can I schedule automatic irrigation?

Yes, you can set custom irrigation schedules via dashboard settings.

How accurate are the sensor readings?

They provide more than 90% accuracy based on calibration.

Can I receive irrigation alerts on mobile?

Yes, you will receive irrigation alerts via SMS and mobile in the Agriverse app.

Crops & Advisory

What crops does AGRIVERSE support?

What is Crop Intelligence?

AI-based system that gives crop recommendations.

What is the Farmer Advisory page?

It provides task and crop suggestions tailored to field conditions.

Does AGRIVERSE recommend based on weather?

Yes, by analysing temperature and rainfall forecasts.
